Easy O2 App

ABOUT THE PROJECT

This project aims to overcome one of the toughest problems faced during the pandemic i.e the supply and proper management of resources especially oxygen cylinders and ventilators. The second wave is a grim reminder that how the lack of a proper management system can result in loss of countless lives, and that was one of the motivations for this project.

HOW THIS PROJECT AIMS TO HELP

This project aims to provide an all in one solution for managing and supplying oxygen cylinders and ventilators to patients in need. By maintaining secure and integrated tables, we are able to provide seamless communication for management as well as supply of these covid critical resources.
